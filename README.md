# Análise de Desempenho Escolar com Python

## Contexto
Projeto de análise de dados educacionais utilizando dados fictícios de uma escola.

## Objetivo
Apoiar decisões pedagógicas por meio da análise de notas, frequência e desempenho
por turma e disciplina.

## Tecnologias
- Python
- Pandas
- Matplotlib
- Google Colab

## Principais Insights
- Diferenças de desempenho entre turmas
- Variação de resultados entre disciplinas
- Relação entre frequência e desempenho acadêmico

## Status
Projeto finalizado para fins de portfólio.
